# Trace Evidence Report

A Trace Evidence Report documents analysis of small transferred materials such as fibers, glass, soil, paint, hair, pollen, residue, or particles.

## Purpose

Trace evidence supports linkage reasoning between actors, locations, objects, vehicles, and events.

It is often subtle and works best as corroboration rather than a standalone solution.

## When to use

Use a Trace Evidence Report when the case depends on:

- contact between people or objects
- movement between locations
- vehicle or tool linkage
- environmental context
- staged scene detection
- transfer evidence
- industrial or occupational materials

## Typical investigative roles

| Role | Suitability |
|---|---|
| Corroborator | Very high |
| Context-dependent clue | High |
| Core clue | Medium |
| Red herring | Medium |
| Contradiction source | Medium |
| Eliminator | Medium |

## Typical evidence exposure

A Trace Evidence Report may expose:

- material type
- source comparison
- location of recovery
- transfer pattern
- rarity or commonality
- environmental match
- inconclusive similarity
- contamination risk
- occupational linkage

## Realism constraints

Trace evidence SHOULD usually be probabilistic or comparative, not absolute.

The report SHOULD distinguish consistency from proof of source.

Specialized interpretation SHOULD be supported by context clues when needed.

## Required metadata

A specification SHOULD include:

- report ID
- sample IDs
- material category
- collection location
- comparison samples
- analyst
- analysis method
- result category
- limitation statement

## Rendering guidance

Trace evidence reports MAY include:

- sample table
- comparison table
- microscopy or lab notes
- conclusion section
- limitation statement
- analyst signature

## Validation checks

- Does the report distinguish similarity from identity?
- Are samples consistent with the Evidence Graph?
- Does interpretation require context clues?
- Are contamination issues intentional if present?
- Is the evidence too decisive for its type?
