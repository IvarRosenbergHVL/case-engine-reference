## V1 role

These document types define reusable forensic artifact functions.

They should describe what a document type can plausibly contain, not a specific case solution.

## Audit note

Forensic document type specifications SHOULD avoid pairing a technical finding with a concrete personal relationship, motive, or solution chain unless the wording is generic and necessary to define the document type.
