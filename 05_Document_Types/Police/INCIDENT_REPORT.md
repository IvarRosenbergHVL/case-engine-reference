# Police Incident Report

A Police Incident Report is an official record of the initial reported incident and first response.

## Purpose

The incident report anchors the case.

It establishes what authorities initially believed happened, who reported it, when it was reported, where it happened, and what immediate actions were taken.

## When to use

Use an Incident Report when the case needs:

- an official starting point
- an initial timeline anchor
- a first version of events
- named responding officers
- initial witness or reporter information
- procedural realism

## Typical investigative roles

| Role | Suitability |
|---|---|
| Anchor | Very high |
| Core clue | Medium |
| Contradiction source | Medium |
| Red herring | Medium |
| Context provider | Low |
| Atmosphere | High |

## Typical evidence exposure

An Incident Report may expose:

- discovery time
- report time
- location
- initial condition of scene
- first named witnesses
- initial suspect references
- missing objects
- immediate inconsistencies
- procedural omissions

## Realism constraints

The report SHOULD only include what responders knew or recorded at that time.

It SHOULD NOT include later forensic conclusions unless added as clearly marked supplements.

It SHOULD use official, concise, procedural language.

## Required metadata

A specification SHOULD include:

- incident number
- reporting officer
- report date and time
- incident date and time
- location
- reporting party
- involved persons
- initial classification
- document status

## Rendering guidance

Incident reports are usually structured, official, and form-like.

They MAY include:

- header with police district or department
- incident number
- typed fields
- narrative section
- signatures or initials
- filing marks

## Validation checks

- Does the report contain only information available at reporting time?
- Does it establish a useful starting frame without solving the case?
- Is the language official rather than literary?
- Are timestamps internally consistent?
- Are later conclusions clearly excluded or marked?
