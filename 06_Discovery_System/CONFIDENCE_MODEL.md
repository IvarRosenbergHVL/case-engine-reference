# Confidence Model

The Confidence Model describes how strongly a player group can justify a hypothesis or conclusion from the archive.

## Purpose

Players may suspect many things, but a satisfying solution requires justified confidence.

The Confidence Model helps distinguish weak suspicion from supported explanation.

## Definition

Confidence is the degree to which available player-facing evidence supports a hypothesis, interpretation, elimination, or solution insight.

Confidence is evidence-relative, not truth-relative.

A correct conclusion with weak evidence is still poorly supported.

## Confidence levels

| Level | Description |
|---|---|
| Speculative | Possible but weakly supported. |
| Plausible | Supported by some evidence but fragile. |
| Supported | Supported by multiple evidence points. |
| Strong | Supported by independent evidence and few unresolved conflicts. |
| Justified | Strong enough to include in final solution reasoning. |

## Confidence effects

Evidence may:

- increase confidence in a hypothesis
- decrease confidence in a hypothesis
- shift confidence from one hypothesis to another
- increase uncertainty by creating a contradiction
- confirm an existing interpretation
- reframe earlier evidence

## Confidence and difficulty

Easy cases may allow confidence to rise steadily.

Hard cases may involve confidence shifts, competing theories, and delayed confirmation.

A hard case should still allow justified confidence by the end.

## Normative requirements

The intended solution SHOULD reach justified confidence from player-facing materials.

Major wrong hypotheses SHOULD remain below justified confidence after all relevant evidence is considered.

A case SHOULD NOT require players to guess between equally supported incompatible solutions.

Confidence SHOULD be based on evidence, corroboration, and contradiction resolution.

## Validation questions

- Can players justify the final explanation?
- Are wrong hypotheses weaker after all evidence is considered?
- Does the case leave multiple incompatible solutions equally supported?
- Does confidence rise through evidence rather than facilitator assertion?
- Is uncertainty intentional or accidental?
