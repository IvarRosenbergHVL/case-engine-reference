# Visual Continuity

Visual Continuity ensures that all artifacts in a case package appear to belong to the same case world.

## Purpose

A case package may contain many generated, human-made, rendered, scanned, and edited artifacts.

Without continuity control, the package can feel like a random collection rather than a coherent investigation archive.

## Definition

Visual Continuity is the consistency of visual language, document style, period, institutional identity, artifact aging, typography, layout, and media treatment across a case package.

## Continuity dimensions

| Dimension | Description |
|---|---|
| Case identity | Shared visual identity for the case package. |
| Institutional identity | Logos, letterheads, stamps, departments, and official marks. |
| Document style | Consistent templates, layout, typography, and metadata treatment. |
| Period consistency | Artifacts fit the intended time period. |
| Media realism | Photos, scans, screenshots, and printed artifacts feel plausible. |
| Aging and wear | Damage, paper texture, compression, and marks are consistent. |
| Color and tone | Palette and contrast choices align across package. |
| Asset reuse | Logos, maps, stamps, and marks remain stable. |

## Continuity risks

Visual continuity may fail when:

- each artifact is generated independently
- logos or stamps vary across documents
- typography changes without reason
- photo aesthetics vary wildly
- screenshots use incompatible device styles
- aging effects are inconsistent
- maps and layouts use unrelated design languages
- AI tools add unplanned visual details

## Normative requirements

A case SHOULD define a Case Style Guide before large-scale artifact production.

Artifact Production Specifications SHOULD reference relevant continuity requirements.

Visual assets that represent the same entity SHOULD remain consistent across artifacts.

Visual continuity SHOULD be checked before release packaging.

## Validation questions

- Does this artifact look like it belongs to the same case package?
- Are repeated logos, stamps, maps, and marks consistent?
- Does the style match the artifact's source and period?
- Did production introduce unplanned visual details?
- Does continuity support realism without hiding evidence?
