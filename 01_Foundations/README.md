## V1 role

The Foundations section establishes the core assumptions behind CER:

- a case is an information system
- documents are views of underlying information
- player-facing evidence must support fair reasoning
- specialized knowledge needed for solving must be embedded in the case package
- the specification describes reusable structures rather than concrete stories
