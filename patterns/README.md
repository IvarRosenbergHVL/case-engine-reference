# Patterns Index

Patterns describe reusable mystery-design structures.

Pattern IDs use the format:

```text
PAT-0001
```

## Pattern categories

- hidden identity
- mistaken motive
- false timeline
- delayed discovery
- opportunity collapse
- reverse alibi
- red herring
- evidence mirror
- missing attachment
- context clue

Patterns are non-normative until connected to rules or tests.
