# Evidence Redundancy

Evidence Redundancy defines how critical information is supported by multiple independent exposures.

## Purpose

Players miss clues, misread documents, split attention, and form premature theories.

Redundancy makes a case fair without making it trivial.

## Definition

Evidence Redundancy is the presence of multiple independent evidence exposures that support the same critical fact, inference, or elimination.

## Redundancy is not repetition

Repeating the same clue in the same form is weak redundancy.

Strong redundancy gives players more than one path toward the same inference.

## Redundancy types

| Type | Description |
|---|---|
| Source redundancy | Different sources support the same inference. |
| Format redundancy | Different document or asset formats expose the same fact. |
| Reasoning redundancy | Different reasoning paths lead to the same conclusion. |
| Temporal redundancy | Evidence appears at different stages of discovery. |
| Context redundancy | Context appears in more than one natural place. |

## Strong redundancy examples

- A witness statement suggests timing, while a log constrains the same timing window.
- A letter suggests motive, while a financial record shows objective pressure.
- A photo shows an object, while an inventory record proves custody.
- A context article explains a technical clue, while a workplace note localizes that knowledge to a suspect.

## Weak redundancy examples

- The same sentence copied into two documents.
- Two documents from the same source saying the same thing.
- A facilitator solution restated in a player document.
- A clue repeated without changing source, format, or inference path.

## Normative requirements

Critical facts SHOULD have at least two independent evidence exposures.

Core solution logic SHOULD NOT depend on a single fragile clue.

Redundancy SHOULD preserve discovery quality by avoiding overly direct repetition.

## Validation questions

- Which critical facts have only one exposure?
- Are exposures independent?
- Do redundant clues appear in different formats or sources?
- Does redundancy make the solution obvious too early?
